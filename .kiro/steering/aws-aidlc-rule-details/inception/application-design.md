# Application Design

Inception stage rules for application design.

## Design Principles

- Follow SOLID principles
- Design for scalability
- Consider security from the start
- Plan for observability and monitoring

## Architecture Patterns

- Choose appropriate architectural patterns
- Document design decisions and rationale
- Consider trade-offs and alternatives
- Align architecture with requirements

## Design Documentation

- Create clear architecture diagrams
- Document component interactions
- Specify data flows and storage
- Define APIs and interfaces
