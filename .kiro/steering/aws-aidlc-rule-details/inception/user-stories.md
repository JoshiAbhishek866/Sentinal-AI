# User Stories

Inception stage rules for user stories.

## User Story Format

As a [user type], I want [goal] so that [benefit].

## Acceptance Criteria

- Define clear acceptance criteria for each story
- Use Given-When-Then format when appropriate
- Ensure stories are testable
- Keep stories small and focused

## Story Quality

- Stories should be independent
- Stories should be negotiable
- Stories should be valuable
- Stories should be estimable
- Stories should be small
- Stories should be testable (INVEST criteria)
