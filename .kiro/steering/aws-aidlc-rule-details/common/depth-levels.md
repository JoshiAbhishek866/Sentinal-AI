# Depth Levels

Shared rules for depth levels across all AI-DLC stages.

## Depth Level Guidelines

- **Level 1**: High-level overview, executive summary
- **Level 2**: Detailed specifications, technical design
- **Level 3**: Implementation details, code-level documentation
- **Level 4**: Deep technical analysis, optimization details

## When to Use Each Level

- Use Level 1 for stakeholder communication
- Use Level 2 for development team planning
- Use Level 3 for implementation guidance
- Use Level 4 for performance tuning and troubleshooting
