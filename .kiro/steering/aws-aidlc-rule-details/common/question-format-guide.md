# Question Format Guide

Shared rules for question formatting across all AI-DLC stages.

## Question Types

1. **Clarification Questions**: To understand requirements better
2. **Design Questions**: To explore architectural options
3. **Implementation Questions**: To resolve technical details
4. **Validation Questions**: To confirm understanding

## Best Practices

- Ask specific, focused questions
- Provide context for each question
- Offer multiple choice options when appropriate
- Prioritize critical questions first
