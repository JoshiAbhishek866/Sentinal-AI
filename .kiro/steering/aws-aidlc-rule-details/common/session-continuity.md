# Session Continuity

Shared rules for maintaining session continuity across AI-DLC stages.

## Session Management

- Maintain context across multiple interactions
- Reference previous decisions and rationale
- Track changes and evolution of requirements
- Document key decisions and their impact

## Context Preservation

- Save important artifacts (requirements, designs, decisions)
- Link related documents and discussions
- Maintain traceability from requirements to implementation
- Update documentation as the project evolves
